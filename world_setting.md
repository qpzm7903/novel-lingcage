# 世界观设定集 (World Setting Bible)

---

## 一、主世界：灵笼（Spirit Cage）

### 时代背景
近未来，地球遭遇未知的"玛娜生态"（Mana Ecology）灾难性爆发。大地被一种奇异的生态体系覆盖，空气中弥漫着玛娜孢子。原本的文明社会彻底崩塌，人类数量锐减95%以上。

### 灯塔（Lighthouse）
- **概述**：人类最后的空中堡垒，一座巨大的浮空城市，依靠反重力技术和能源核心悬浮在空中，远离地面的玛娜生态威胁。
- **社会结构**：严格的基因等级制度。
  - **上民（Upper Citizens）**：基因等级高，享受优质资源、教育、居住环境，担任指挥官、科学家、精英战士等核心职务。
  - **尘民（Dust Citizens）**：基因等级低，被当作劳动力使用，居住环境恶劣，食物配给有限，没有生育权和上升通道。
- **三大法则**：灯塔统治核心——"严控人口、基因优先、杜绝情感"。
  - 禁止恋爱、禁止家庭关系、禁止任何形式的"多余情感"。
  - 所有婚配由基因匹配系统决定，仅以繁衍优质基因为目的。
- **猎荒者小队**：灯塔最精锐的地面作战部队，负责降落地面搜集物资和能源。每次出航都面临噬极兽的致命威胁。每支猎荒者小队约12名战斗人员（分3个战术小组）+ 若干后勤支援人员，佩戴银色羽翼猎荒者徽章。
- **监控等级**：从低到高分为普通、二级关注、一级关注、高危。林墨在第9章被提升至"二级关注"，第14章提升至"一级关注"。
- **特殊编制**：城主直辖的独立作战单位，代号以元素命名。林墨成为"灰烬"（Ashes），编号S-0001。
- **基因中心**：位于灯塔上层区域，由首席技术官负责。配备深层基因序列分析能力，可检测骨髓层面异常。

- **地面已知区域**
- **G-07**：废弃工业区，设有化工厂，是第7-8章采集任务的坐标点。行动中遭遇九只镰刀兽袭击。
- **石化冶炼厂**：六座反应塔的工业废墟，占地广阔，外壁被紫色玛娜藤蔓覆盖。第12章远征目标之一，存在镰刀兽群和罕见的声波型噬极兽"咆哮者"。远征中成功采集冷却剂原料。
- **废弃生物实验室**：半沉入地下的科研园区，"生命科学研究院生态适应部"所在地。入口被混凝土堵塞。内部主楼保存相对完好，有组织结构图揭示灾难前研究课题。内部有兽王幼体潜伏。第13章远征探索目标，林墨在此获得关键数据硬盘。
- **生命科学研究院（推测位置）**：地图上标注为"旧世界遗迹·未勘测"的区域，兽王迁徙路线的最终指向。兽王呈扇形排列、经过三个高价值资源点后指向此地。可能为"母体"所在地。
- **旧时代通信基站**：远征计划中的第三目标，用于建立地面临时哨站。
- **北部死寂区域（第101-110章探索）**：绿洲以北约五天步行距离的异常区域。地面呈黑灰色颗粒状沙化态，无任何生命迹象，玛娜网络被整片"切断"。中心位置为第七号全球生态监测站，以北十二公里处有旧时代"地壳深层采样计划"钻孔封盖（DSDP-07-3KM），封盖下约三千米处封印着一个维度夹层中的古老存在。
- **第七号全球生态监测站**：位于死寂区域中心，半掩埋地下的旧时代建筑。入口走廊刻满被斜线划穿的圆。主控制室中一台服务器自灾难纪元后第二年连续运行约十八年未关闭——维持三个辅助通道（监听/喂养/回应）。发现值班员H约四百天手写日记和阿福发现的手绘五节点地图。
- **第五节点·林远山私人实验室（第114-116章探索）**：位于绿洲以东约三百公里，坐标与霍恩灯塔西北约八十公里处重合。被约三十米高的灰白色矿物质粉尘浓雾（灰域）包围，雾中玛娜浓度为零、共生原力完全失效。中心为黑色合金围墙（高约十米，特殊能量场阻隔灵视——与管理员之印暗金纹路有微弱共鸣）。内部分三层：一层为跨维度能量稳定性研究区，大量林远山手写笔记（灾难纪元前六年至前三个月）；二层为生物培养区，六个培养舱已停运，六号舱标签"SR-01·基因序列·第三备份·激活状态：待定"；三层为林远山私人办公室，有管理员之印感应锁。办公室内发现林远山笔记本（最后记载指向沈若华）、林远山与沈若华的合影相框、沈若华身份牌、银色数据晶体（办公桌暗格）。三层另有紧急逃生通道通往北侧约两百米处废弃泵站。
- **第四节点·南境天然维度薄弱点（第118章侦察）**：绿洲以南约一百五十公里处，坐标与暗色碎片投射的维度坐标高度重合。地面为普通玛娜植被覆盖的丘陵，无任何人工建筑。地下约二十米深处存在直径约十米的天然球形空腔，内壁覆盖晶化物质（与灵修世界裂缝洞穴的晶化层一致）。为五个维度薄弱点中唯一纯天然未开发状态的节点。林墨在空腔正上方岩石中以灵元刻入感知印记用于远程监控。

- **灯塔内部区域（新增）**
- **旧舱段H-09**：灯塔底部废弃区域，连尘民都不再使用，走廊狭窄低矮布满锈蚀。内有一处约四十平方米的隐蔽据点，被火种改造为秘密集会场所。具有弧形天花板，隐藏式密码锁舱门。
- **基因中心**：灯塔上层区域，走廊宽阔洁白，空气中弥漫消毒水气味。拥有深层基因序列分析能力（骨髓级）。首席技术官常驻于此。
- **指挥区顶层**：灯塔最高层，城主办公室所在地。简洁空间，仅有一张金属办公桌、数据终端和一幅旧世界城市夜景油画。拥有多重安检门。

- **噬极兽（Chimera / Source Beasts）**
- 由玛娜生态催生的恐怖生物，形态各异，从数米到数十米不等。
- 拥有强大的再生能力和对人类的捕食本能。
- 会吸收人类的"生命源质"（一种类似灵魂能量的本质）。
- 弱点：头部神经节——再生中枢的薄弱点。
- 等级体系：普通噬极兽（如镰刀兽）→ 声波/特殊型噬极兽（如咆哮者）→ 高级噬极兽（如第7章出现的装甲巨兽，体长超二十米，背部覆盖厚重甲壳，甲缝透暗红光）→ 兽王幼体（未成年体，体长约五米，已具备战术意识）→ 兽王（第10章预告，三只正在向灯塔迁徙，体长超三十米，移动的山脉级存在）→ 未知的更高存在（"母体"假说）。

**镰刀兽**——最常见种类，体型如大型猛兽、前肢如镰刀，群体行动，弱点为颈部神经节。

**咆哮者（Roarer）**——罕见的声波型噬极兽（第12章首次登场）。体型如卡车，身体呈多种生物组织粗暴拼接状。胸腔有一巨大半透明囊状结构，充盈荧光绿色气体。没有嘴巴，发音器官为腹部的管状器官，可发射神经攻击性音波，导致人类剧烈头痛、失去战力甚至七窍流血。弱点：腹部的发声管。被晨曦在第12章斩断后击杀。

**兽王幼体**——年轻的兽王个体（第13章首次登场）。体长约五米，深紫色外骨骼表面油亮，背部有暗红色尖刺。速度远超镰刀兽，外骨骼防御极高（A级基因战士全力一击仅留浅痕）。具有战术意识，会主动制造混乱后攻击弱点。弱点：关节连接处的缝隙（腋下、后腿根部）。第13章由林墨发现弱点并与晨曦合力击杀。

**雾隐兽（Mist Beast）**——特殊型噬极兽（第21章首次登场）。体型如大型犬科动物，体表覆盖半透明薄膜状组织，能持续释放干扰感知的孢子雾。孢子雾可压制斗气感知（从十几米锐减至不足三米），同时干扰视觉和距离判断。行为模式与普通噬极兽不同——更有耐心，更像在"侦察"而非"捕食"，疑似受母体低频信号直接指挥。弱点：腹部释放孢子的器官。第23章由晨曦在听觉锁定下一刀贯穿，林墨补枪命中后腿关节后合力击杀。

**强化镰刀兽（半兽王级）**——普通镰刀兽的强化个体（第28章首次登场）。体长约八米，深紫色外骨骼，背部竖立暗红色尖刺。防御力极高——A级基因战士全力一击仅留浅痕。弱点：关节连接处缝隙（腋下、后腿根部），与兽王幼体相同。第28章在聚居地防御战中出现，晨曦利用弱点攻击后负伤撤退。

### 玛娜生态
- 覆盖地面的紫色/蓝色发光植被体系。
- 会主动扩散孢子，侵蚀非适应区域。
- 与噬极兽构成共生关系——植被提供能量，噬极兽提供保护。
- 玛娜植物有触觉感知和群体信息传递能力（第7章发现：触碰一株藤蔓，十米外另一株的卷须同步震颤）。
- 地面存在完整的变异生物链（变异昆虫如背带蓝光的黑色甲虫、变异爬行类如四足蜥蜴状生物等）。

### 旧世界遗迹
- 地面废墟中发现的"生命工程"铭牌（第7章），带有三环树形标志和编码L-07-G2-113，暗示玛娜生态爆发前存在相关研究。
- 生物实验室中发现的"生命科学研究院生态适应部"组织结构图，研究课题包括"玛娜化环境中的跨物种适应"和"共生可行性评估"。最后一条研究记录在灾难纪元前六个月。

### 玛娜生态新发现（第111-120章）
- **神秘声音的族群本质**（第120章霍恩揭示）：神秘声音与北部三千米钻孔中的存在、灵修世界裂缝中的存在为同一"族群"的三个独立个体——维度之间的流浪者。它们不属于任何维度世界，而是在维度夹缝中漂流的存在。三个个体共享同一种本质但各有独立意识：一个被灵修世界古代修士封印在裂缝之下；一个被旧时代钻探项目意外凿穿封印后困在玛娜世界三千米深处；第三个以无形状态在维度夹层中漂流，找到了霍恩并建立了寄生线系统和候选体网络。
- **"回家"诉求的真伪**：北部三千米存在向林墨传递"想回家"的信息，但霍恩指出"它们对谁都是这个说法"。神秘声音族群的真实目的尚不明确，但已知它们在利用人类作为"开门的人"——它们需要拥有身体的存在来操作维度通道。
- **暗色碎片觉醒记忆载体功能**（第117章）：暗色碎片接触林远山笔迹后激活内部裂纹网络，投射出三维维度坐标图（指向第四节点）。证明碎片不仅是"门体残渣"，更是林远山故意留在灵渊秘境裂缝底部的"记忆载体"——记录着维度薄弱点的空间关系。碎片在读取了林远山笔记本信息后变得更加"活跃"（温度升高，裂纹中透出暗金色微光）。
- **五节点网络本质**（第117章林墨推论）：以五个维度薄弱点为基座的封印/通道系统——1号北部钻孔（人工凿穿）、2号沿海档案库（生命工程标准设施）、3号西山通讯站（寄生线中继站）、4号南境天然薄弱点（未开发）、5号东境实验室（被改造的半成品）。林远山布下此网络用于监测维度薄弱点变化，神秘声音利用了它进行渗透。五节点是不完整的五边形——碎片中还记录有未被激活的额外纹路，尚需更多信息才能完全显现。
- **沈若华身份揭露**（第115章）：前生命工程·伦理审查委员会·独立观察员。灾难纪元前四个月从记录中消失——取走林远山留在第五节点六号培养舱的第三份基因序列备份（"SR-01·基因序列·第三备份·激活状态：待定"），成为"不在系统中、不被任何人监控"的最终防线。容貌与苏若兰有七八分相似（名字只差一字），暗示存在亲属关系。最后行踪指向南方极远处，超出五节点地图范围。
- **银色数据晶体**（第115章发现）：林墨在第五节点三层办公室桌下暗格发现的未读取数据晶体。生命工程标准规格但型号非通用——银白色金属外壳，无接口无指示灯，拇指大小。与沿海遗址保险库中的蓝色数据晶体设计相似但颜色不同。林墨暂未读取。
- **"第五个月亮下沉时"之谜**（第115章）：沈若华留在第五节点办公室桌垫下的字条内容。目前无法解读其指向含义。林墨将字条贴身收藏。
- **灰域**（第114章首次遭遇）：第五节点周边约三十米高的灰白色浓雾墙，由粒径不到三微米的硅酸盐矿物质粉尘悬浮形成。雾中玛娜孢子浓度为零，共生原力完全失效（信号被"遮蔽"而非北部死寂区域的"切断"）。管理员之印在灰雾中不发热不发光，灵元感知被压缩至不到十米。定向极度困难。
- **第五候选体·直接被神秘声音占据**（第116章首次出现）：候选体中第五个个体——灵魂频率完整无拼接裂缝，非远程操控而是神秘声音以意识降临直接"住进"了这具身体。动作流畅自然如身体"本来就是她的"。不追击已暴露的林墨，确认其拿到线索后即撤离。神秘声音故意让林墨取得笔记本和身份牌——它希望林墨顺着线索南下找沈若华。
- **霍恩切断连接成为第三方**（第118-120章）：霍恩主动切断与神秘声音的一切连接（所有通讯线路物理切断）。真实动机为纯粹的利益重新计算——发现被当作"用完可丢的工具"后拒绝接受此身份。认为林墨是唯一可能制衡神秘声音的力量，主动提供沈若华坐标数据芯片和一份原始管理员基因培养皿样本作为信物，自余三份作为保险。目前处于完全独立的第三方状态。"我从来没想过拯救人类。我只是不想输给一个连身体都没有的东西。"
- **三方格局形成**（第120章）：林墨方——掌握五节点信息、管理员之印、暗色碎片、林远山笔记本；神秘声音方——掌握候选体网络、寄生线系统、第五节点控制权，正寻找沈若华；霍恩方——独立第三方，掌握三份剩余培养皿和二十年积累的底牌。林墨决定南下寻找沈若华——既非急于决战也非结盟，而是去取林远山留下的最后一枚砝码。
- **南方同源能量波动**（第120章程末）：母体感知到南方极远处传来的能量波动——与北部三千米钻孔中的存在气息同源但更加稳定安静，与霍恩提供的沈若华坐标方向一致。
- **章末暗线·南方某地**（第120章）：一只女性的手将一枚旧时代硬币收进抽屉，落在其他同样被磨损的硬币中间。暗示沈若华（或与该路线相关的关键人物）在南方持续活动至今。

### 第121-130章新增设定

- **银色数据晶体读取**（第121章）：以管理员之印+灵元+斗气三重力量同时触碰可激活晶体表面纹路，林远山的录音直接传入意识。晶体编码方式为沈若华自创——月相+星位+地平线夹角三者交叉定位。晶体与暗色碎片靠近时产生微弱共振——暗金色光芒向晶体方向流动。晶体需在特定位置（LP-07）月光下才能投射完整坐标路径，是"定位锁的钥匙"而非钥匙本身。

## 二、金手指设定：万界穿越（Myriad World Traversal）

### 能力本质
主角觉醒的独特天赋——意识（后期可带肉身）穿越到其他小说/影视/游戏世界。每个世界的时间流速与主世界不同，初期设定为 **10:1**（异世界10天，主世界1天），但第四世界灵修世界为 **20:1**（更慢的时间流速）。

### 第181-190章新增设定

- **东部废土地貌**（第185-187章）：绿洲以东的地貌分多层过渡——灰褐色沙砾→黄褐色硬土→灰白色贝壳碎屑砂质土。玛娜植被密度随远离母体核心区域而下降直至完全消失。土壤中无玛娜根系，空气中无玛娜孢子。空气湿度高于西部废土，可闻到海风咸味。沿海城镇因无玛娜侵蚀，建筑保存完好——墙体仅自然风化和盐雾腐蚀，泛白色盐霜。存在非玛娜生态的原生动物（灰白色皮毛如小牛的犬科生物）——在玛娜灾难前就存在的幸存物种。
- **滨海新区·生命工程·第七气象观测站**（第183章发现）：林远山在五节点系统和E-04之外独立建立的第六个设施。盒盖内侧刻字指向此位置。林远山在此设施地下建造了小型实验室并存放了《关于旁观者印记的完整研究报告》。远离玛娜生态覆盖范围，通讯设备约十五年前彻底中断。
- **PRC-07海洋观测平台**（第190章）：位于岛屿以东约七到十公里海面上的旧时代气象海洋监测平台。锈蚀严重但结构完好，甲板室顶部小型风力发电机持续运转二十年产生电力。甲板室内有一台运转中的信号发射器连接深循环蓄电池——"第七节点"信号的实际发射源。发射器旁有打印纸条和录音笔——"第三方"在灾难纪元前一年就已部署。
- **第七节点信号真相**（第189-190章揭示）：信号从林远帆到达岛屿第一天就开始播报——但不是他发的。实际发射源来自PRC-07平台的自动发射器，由"第三方"在灾难纪元前一年设置。
- **第零号节点**（第189章林远山手稿揭示）：林远山在第七气象观测站选址时通过深层钻探发现的地下结构——位于地下八百到一千二百米处。放射性测年超过四万年。墙壁垂直、转角直角、表面有纹路——不是地质运动产物，是有意建造的。比生命工程所有节点更古老，林远山在建造管理员基因前就知道它存在。管家者印记指向的方向可能与此结构有关。
- **第九节点**（第190章预告）：维度夹层中的未知位置——不在陆地不在海洋。"第三方"自称在第九节点等待管理员继承者。林远山临终前警告"不要相信从第九节点回来的任何人"。
- **"第三方"存在**（第190章首次出现）：自称从灾难纪元前一年就开始等待的存在。认识林远山和旁观者印记。在PRC-07平台留下纸条和录音笔，自称"不在林墨一边也不在霍恩一边"。声音特征为通过空气振动直接传播——与汐和北存在相似但更古老更缓慢。署名符号与万界之门最底层符号一致——"沉默"/"在此之前"/"原始"。也联系了霍恩——声称用林远山基因制作了旁观者印记"副本"。身份完全未知。
- **林远山的完整手稿**（第189章）：《关于旁观者印记的完整研究报告》——五厘米厚的A4打印纸，灾难纪元前六个月写就。四个核心揭示：①旁观者印记是人类基因组中本来就有的非编码"识别标签"结构。②林远山利用它作为骨架建造管理员系统。③林墨不是在"设计"中诞生的——是在旁若者印记引导下自然诞生的继承者。④第零号节点是四万年前的有意建造结构——管理员基因和所有节点只是通往这个更古老东西的台阶。手稿中还包含一枚旧式U盘——数据备份需特定终端读取。
- **旁观者印记"空"点状态**（第184章）：林墨出发前夜以灵元深入管理员之印最核心，发现一个"空"点——不是颜色不是光不是暗而是一种"缺失感"。性质与万界之门左下角同心圆图案中心的凹陷相同。触碰无反应不排斥不交互，如同半梦半醒的存在。在灵视下呈现极微弱"温度差"——比周围低一点点。林墨黎明前第三次触碰时感知到转瞬即逝的"情绪"波动——旁观者有感觉但选择不回应。林墨推论：管理员基因的所有功能建立在旁观者印记基底之上，没有旁观者印记管理员之印根本不会工作。
- **阿福收到"快走"警告**（第190章）：裂缝修补后三十一天深夜，阿福在绿洲监测站异频段（林墨在PRC-07平台期间）捕获一段仅三秒的极短信号。声音从门缝般遥远的位置传来——用中文低语两字"快——走——"。语气非威胁而是警告——如同黑暗中看到什么的人用最后力气发出的提醒。此后该频段再无任何信号。阿福在日志中记录后继续守候。
- **霍恩收到神秘纸条**（第190章）：裂缝修补后霍恩在其地下设施入口气密门外发现无标记金属盒。盒中纸条与PRC-07平台纸条字迹完全一致（宋体字、针式打印机）。内容称霍恩体内只有一半旁观者印记，另一半在"死去的哥哥身上"，并声称用林远山基因制作了旁观者印记副本存放在第九节点。纸条署名与万界之门最底层符号一致。霍恩读完后面无表情折叠收好——他没有一个活着的哥哥。暗示此"哥哥"指代某种未知的存在或镜像意识。
